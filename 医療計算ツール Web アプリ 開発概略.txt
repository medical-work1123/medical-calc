## 医療計算ツール Web アプリ 開発概略

### プロジェクト概要

本セッションでは、iPad・スマートフォン対応の医療計算ツール Web アプリを、単一の HTML ファイル（index.html）として設計・開発し、GitHub Pages による公開までを一貫して行った。

---

### 開発の経緯と経過

**第1フェーズ：基本計算機の作成**

最初に、臨床現場で頻用される基本的な計算機として BMI（肥満度）、CCr（Cockcroft-Gault 式によるクレアチニンクリアランス）、FENa（尿中ナトリウム排泄率）の3つを実装した。レイアウトはタブ切り替え型とし、タッチ操作に適したデザインを採用した。

**第2フェーズ：肝臓スコアの追加**

次に肝臓疾患に関連する計算機として、MELD スコア（末期肝疾患モデル）、MDF スコア（Maddrey 判別関数）、GAHS スコア（Glasgow Alcoholic Hepatitis Score）の3つを追加実装した。

**第3フェーズ：大規模リニューアル（5カテゴリー・15計算機体制）**

計算機の種類が増えたことを受け、ナビゲーション構造を2階層化する大規模なリニューアルを実施した。上段で診療科カテゴリーを選択し、下段でその中の計算機を選ぶ構成に刷新した。あわせて以下の計算機を新規追加した。

- **肝臓**：JAS スコア（アルコール性肝炎診断基準）
- **糖尿病**：BMI、HOMA-IR／HOMA-β（インスリン抵抗性・β細胞機能）、CPR インデックス（β細胞分泌能）
- **脂質異常症**：吹田スコア（10年冠動脈疾患リスク）、LDL 管理目標値（JAS 2022年ガイドライン準拠）
- **電解質**：Anion Gap（補正計算対応）、補正カルシウム、血漿浸透圧（浸透圧ギャップ計算対応）、補正ナトリウム（Katz 法・Hillier 法の2方式）

---

|---|---|---|

## 第2セッション 開発概略

### 概要

前セッションで構築した5カテゴリー・15計算機体制に対し、感染症グループ（7計算機）・膵臓疾患グループ（1計算機）・胆道系疾患グループ（4計算機）を新規追加するとともに、既存の脂質異常症グループの計算機をJAS 2022年版ガイドライン準拠に全面改訂した。あわせて、複数のバグを発見・修正した。

---

### 第4フェーズ：感染症・膵胆道グループの追加（計12計算機追加）

カテゴリーナビゲーションに「感染症」「膵・胆道」の2グループを追加した。

**感染症グループ（7計算機）**

| 計算機 | 内容 |
|---|---|
| Modified Centor スコア | McIsaac修正版。年齢補正（3〜14歳+1点、≥45歳−1点）を含む5項目。GAS確率とRADT/抗菌薬投与の判断基準を表示。 |
| 急性鼻副鼻腔炎 | EPOS 2020準拠。主症状4項目によるARS診断＋VAS重症度分類（軽症/中等症/重症）＋細菌性疑い因子4項目の判定。 |
| A-DROP スコア | 日本呼吸器学会 市中肺炎重症度分類。A（年齢）・D（脱水）・R（呼吸不全）・O（意識障害）・P（血圧低下）の5項目。R/O/P単独陽性時の重症判定を実装。 |
| 肺炎鑑別（細菌性 vs 非定型） | Miyashitaスコア準拠。典型細菌性因子6項目・非定型肺炎因子6項目を対比評価し、細菌性/非定型/鑑別困難を判定。抗菌薬選択指針も表示。 |
| qSOFA | Sepsis-3 quick SOFA。3項目で敗血症スクリーニング。スコア≥2で詳細評価（SOFA・血液培養等）を促す。 |
| SOFA スコア | Sequential Organ Failure Assessment。呼吸・凝固・肝・循環・CNS・腎の6臓器系を各0〜4点で評価（合計0〜24点）。ICU死亡率と敗血症/敗血症性ショックの定義を表示。 |
| SIRS 診断基準 | 体温・心拍数・呼吸数・WBCの4項目。2項目以上でSIRS診断。SIRS→敗血症（旧定義）→敗血症（Sepsis-3）→敗血症性ショックの概念整理表を併記。 |

**膵・胆道グループ（5計算機）**

| 計算機 | 内容 |
|---|---|
| 急性膵炎 重症度スコア | 厚生労働省スコア。予後因子9項目＋造影CTグレード（0〜2点）。合計3点以上で重症急性膵炎と判定。 |
| 急性胆嚢炎 診断基準 | 東京ガイドライン2018（TG18）。A（局所炎症）＋B（全身炎症）＋C（画像）により確定診断/疑い診断/基準外を判定。 |
| 急性胆管炎 診断基準 | TG18。A（全身炎症）＋B（胆汁うっ滞）＋C（画像）により判定。Charcot三徴・Reynolds五徴の解説を付記。 |
| 急性胆嚢炎 重症度判定 | TG18 Grade I〜III。臓器機能障害6項目（Grade III）・局所炎症4項目（Grade II）に基づき分類し、治療方針（手術/ドレナージ）を表示。 |
| 急性胆管炎 重症度判定 | TG18 Grade I〜III。臓器機能障害6項目（Grade III）・高リスク因子5項目（Grade II）に基づき分類。緊急ERCP/PTCDの適応を表示。 |

### 第5フェーズ：バグ修正（3件）

**① setRC 関数のID参照バグ**

結果表示関数 `setRC(id, ...)` が内部で `id + '-rv'` のようにIDを組み立てていたが、HTMLの内部要素IDは `id` から `-rc` サフィックスを除いたものに `-rv` を付加したパターン（例：`ccr-rc` → `ccr-rv`）であった。このため全計算機で数値・解釈文・備考欄が表示されないバグが存在していた。`id.replace(/-rc$/, '')` で基底IDを取り出してから内部要素を参照するよう修正し、全計算機で正常表示されるようになった。

**② チェックボックス ダブルトグルバグ**

`<label>` 要素がチェックボックスを内包する構造では、ラベルをクリックすると ① ラベルの `onclick` ハンドラ（`toggleChk`）が発火し、② ブラウザがチェックボックスへ合成クリックを送信し、③ その合成クリックが親ラベルへバブリングして `toggleChk` が2回呼ばれる問題が発生していた。結果として、チェックのオン/オフが1回の操作でキャンセルされ「チェックが外せない」状態になっていた。以下の3点を修正した。

- 全 `.chk-item` 内のチェックボックスに `stopPropagation` リスナーを追加し、バブリングによる二重呼び出しを防止。
- `toggleChk` を `classList.toggle('sel')` の戻り値（`nowSel`）で `cb.checked` を確定させる方式に変更。
- `isChk(id)` を `cb.checked` プロパティではなく、親 `.chk-item` 要素の `sel` クラスの有無から状態を判定するよう変更（ネイティブの二重トグルの影響を受けない実装）。

**③ チェックボックスの✓ 常時表示バグ**

`.chk-box` のCSSに `color` 指定がなかったため、未選択時も `✓` 文字が常に見える状態になっていた。`.chk-box { color: transparent; }` を追加し、選択時（`.chk-item.sel .chk-box`）のみ `color: #fff` で白表示されるよう修正した。

### 第6フェーズ：脂質異常症グループの全面改訂（JAS 2022準拠）

**① 吹田スコア → 久山町スコアへ置き換え**

従来の「吹田スコア」はJAS 2017相当の実装であり、年齢・性別・喫煙・収縮期血圧・TC・HDL・DM・CKDをスコア変数として年齢非依存のリスク分類を行っていた。JAS 2022ガイドラインに準拠し、一次予防で使用される「久山町スコア」に全面置き換えた。

主な変更点：

| 項目 | 旧（吹田スコア） | 新（久山町スコア） |
|---|---|---|
| 年齢の扱い | スコア変数 | リスク列の決定のみ（スコアに含まれない） |
| 脂質 | 総コレステロール（TC） | LDL-コレステロール |
| CKD | スコア変数 | 含まない |
| 最高点 | — | 19点（男性+7、BP+4、DM+1、LDL+3、HDL+2、喫煙+2） |
| リスク区分 | スコアのみで固定閾値 | **年齢群別の可変閾値** |

年齢別リスク閾値（同スコアでも年齢によりリスク区分が異なる）：

| 年齢 | 低リスク（<2%） | 中リスク（2〜9%） | 高リスク（≥10%） |
|---|---|---|---|
| 40〜49歳 | ≤12点 | 13〜19点 | 到達不可 |
| 50〜59歳 | ≤7点 | 8〜18点 | ≥19点 |
| 60〜69歳 | ≤1点 | 2〜12点 | ≥13点 |
| 70〜79歳 | 到達不可 | ≤7点 | ≥8点 |

10年ASCVD発症推定リスク（%）のルックアップテーブル（スコア0〜19 × 4年齢群）を実装し、スコアに対応する推定リスク値を計算結果に表示するようにした。

**② LDL管理目標値 フローチャートをJAS 2022に全面改訂**

従来の実装はJAS 2017相当であり、「心筋梗塞・狭心症の既往」と「リスク因子数のカウント」による分類であった。JAS 2022「動脈硬化性疾患予防ガイドライン」のフローチャートに完全準拠し、以下のステップ型判定に作り直した。

**STEP 1：二次予防の判定**
- 冠動脈疾患の既往（心筋梗塞・狭心症・PCI・CABG等）
- アテローム血栓性脳梗塞の既往
- いずれかあり → 二次予防へ（STEP 2a）
- いずれもなし → 一次予防へ（STEP 2b）

**STEP 2a：二次予防・超高リスク判定**（ACS・DM合併・CAD再発・多枝/多部位病変・FH合併）
- 該当あり → LDL <70 mg/dL（non-HDL <100）
- 該当なし → LDL <100 mg/dL（non-HDL <130）

**STEP 2b：一次予防・絶対的高リスク判定**（DM・CKD・PAD）
- 該当あり → LDL <120 mg/dL（non-HDL <150）
- 該当なし → 久山町スコアで評価（低/中/高リスクで <160/<140/<120 mg/dL）
